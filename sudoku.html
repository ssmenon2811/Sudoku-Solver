Sudoku Rules

Sudoku is a logic-based number placement puzzle. The objective is to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 subgrids that compose the grid (also called "boxes", "blocks", or "regions") contain all of the digits from 1 to 9.

Each puzzle starts with some cells filled in. These initial digits cannot be changed throughout the course of the game.
